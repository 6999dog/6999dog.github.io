重新捡起c语言（一）

2020年04月19日 2.6k 字 大概 11 分钟

昨天晚上的时候准备写ACM试题，真的绝了
看着题目分析了一下思路，马上就能想到该怎么写
但是我实际写的时候由于一些晓得细节和语法连连报错
从今天开始把C语言捡起来

首先就来谈谈我昨天遇到的问题吧

  1. 在main函数的末尾，我就没有写return 0;
    后来我就去查了一下它的作用
    return 0; 语句用于表示退出程序
  2. 昨天在写字符串的判断的时候直接就写了字符串并没有声明
    char desc = “desc”
    这样之后才没有报错
  3. 在使用指针数组的时候没有开辟空间
    对头文件的使用也王的一干二净

    科普一下：stdio.h 是一个头文件 (标准输入输出头文件) ,
    #include 是一个预处理命令，用来引入头文件。
    当编译器遇到 printf() 函数时，如果没有找到 stdio.h 头文件，
    会发生编译错误
    由此可见对于头文件的引用会影响之后的语句

  4. 最近写多了网页js之类的语句
    习惯了用缩进来控制函数结构，
    很多语句都掉了{}
  5. 发现自己写代码的时候没有注释的习惯，很多时候自己都看不懂自己的代码
    这一点必须马上改过来

正式开始我捡起C语言的行动了

  1. 字符和字符串的表示
    -‘a’单引号里面只能用一个字符
    -“snai”双引号里面可以引用多个字符

  2. 变量的声明
    -extern表示这个变量在其他的地方应用，不需要存储空间
    -例如：

    #include <stdio.h>
    /外部变量声明/
    extern int x ;
    extern int y ;
    int addtwonum()
    {
    return x+y;
    }

  3. 定义常量
    -使用#define在头文件下面声明
    -使用const关键字在数据前面声明 const int x =10;

  4. 运算符
    -刚刚在运算符里面看到了，我一直以为是单纯的指针
    -    运算符的作用就是指向一个变量
    -后续在指针里面继续思考

  5. 循环
    -其实循环这个没有什么好说的
    -但是我觉得最重要的就是自己的思维要清晰
    -控制好循环的次数，不要陷入死循环

  6. 函数
    -这一块带给我的疑惑就是传值的方式问题
    -我绝大多数的时候都是直接使用了传值调用的方法
    -我不会用引用方式传入函数，这里放个例子吧
    -后续的过程中我会一直加油练习编程，用实操来锻炼自己

    #include < stdio.h >
    /* 函数声明 /
    void swap(int *x, int *y);
    int main ()
    {
    /     局部变量定义 */
    int a = 100;
    int b = 200;
    printf(“交换前，a 的值： %d\n”, a );
    printf(“交换前，b 的值： %d\n”, b );
    /调用函数来交换值
    &a 表示指向 a 的指针，即变量 a 的地址
    &b 表示指向 b 的指针，即变量 b 的地址
    /
    swap(&a, &b);
    printf(“交换后，a 的值： %d\n”, a );
    printf(“交换后，b 的值： %d\n”, b );
    return 0;
    }

  7. 数组
    -这里其实还是指针的事，通过指针来传递数组
    -还有熟悉如何定义数组
    -如果希望可以得到一个返回数组的函数，就要声明一个指针的函数
    -这里还要补充的就是关于指向数组的指针

    声明一个指针  int *P;
声明一个数组  int double[5];
那么如何用指针来控制数组输出呢

    p=double;
    *(double+0)
    *(double+1)
    *(p+0)
    *(p+1)
    可以采用这样的方式进行输出数组元素

  8. 枚举
    -关键字：enum
    -对于枚举给我的感受就是定义起来像数据结构，使用起来像数组

  9. 指针
    -指针是一个变量，其值为另一个变量的地址
    -如何使用指针？
    -定义一个指针变量、把变量地址赋值给指针、访问指针变量中可用地址的值
    -声明一个指针p，p就是代表存储地址，*p就是值
    -声明一个变量p，&p才是存储地址，p是值
    -声明一个指针ptr,ptr++就是指向下一个地址
    -在传递函数的时候指针的参数要写变量的地址
    -函数指针是指向函数的指针变量。
    -通常我们说的指针变量是指向一个整型、字符型或数组等变量，而函数指针是指向函数。
    -函数指针可以像一般函数一样，用于调用函数、传递参数。

    #include < stdio.h>
    int max(int x, int y)
    {
    return x > y ? x : y;
    }
    int main(void)
    {
    /* p 是函数指针 /
    int (     p)(int, int) = & max; // &可以省略
    int a, b, c, d;
    printf(“请输入三个数字:”);
    scanf(“%d %d %d”, & a, & b, & c);
    /* 与直接调用函数等价，d = max(max(a, b), c) */
    d = p(p(a, b), c);
    printf(“最大的数字是: %d\n”, d);
    return 0;
    }

  10. 字符串
    -在C语言中，字符串实际上是使用 null 字符 ‘\0’ 终止的一维字符数组。
    -char greeting[6] = {‘H’, ‘e’, ‘l’, ‘l’, ‘o’, ‘\0’};
    -char greeting[] = “Hello”;
    ps:我觉得C语言跟其他的语言来说，规范真的超级重要
    -strcpy(s1, s2);//复制字符串 s2 到字符串 s1。
    -strcat(s1, s2);//连接字符串 s2 到字符串 s1 的末尾。
    -strlen(s1);//返回字符串 s1 的长度。
    -strcmp(s1, s2);//如果 s1 和 s2 是相同的，则返回 0；如果 s1 < s2 则返回小于 0；如果 s1>s2 则返回大于 0。
    -strchr(s1, ch);//返回一个指针，指向字符串 s1 中字符 ch 的第一次出现的位置。
    -strstr(s1, s2);//返回一个指针，指向字符串 s1 中字符串 s2 的第一次出现的位置。

    -关键词 struct

    在访问其成员的时候要用到->访问;

  12. 位域
    -这个是我很陌生的区域，因为之前没有遇到过
    -位域的定义跟结构的定义差不多，在使用的时候需要在变量的后面：（长度）
    -使用位域的好处就是节省了空间
    -位域依旧有指针形式（无处不在的指针）

  13. 共用体
    -关键词 union
    -共用体是一种特殊的数据类型，允许您在相同的内存位置存储不同的数据类型
    -其定义形式跟结构体差不多

  14. typeof
    -C 语言提供了 typedef 关键字，您可以使用它来为类型取一个新的名字
    -typeof和#define的区别
    #define 是 C 指令，用于为各种数据类型定义别名，与 typedef 类似，但是它们有以下几点不同：
    typedef 仅限于为类型定义符号名称，#define 不仅可以为类型定义别名，也能为数值定义别名，比如您可以定义 1 为 ONE。
    typedef 是由编译器执行解释的，#define 语句是由预编译器进行处理的。

    -在字符串中有getchar()和putchar()
    -gets()和puts()

  17. 预处理器
    -这里有一些C语言自带的
    -我们最常用的就是定义宏

  18. 头文件
    -头文件是扩展名为 .h 的文件，包含了 C 函数声明和宏定义，被多个源文件中引用共享。有两种类型的头文件：程序员编写的头文件和编译器自带的头文件。
    我一直以为头文件的引用没有区别
    #include < > 引用的是编译器的类库路径里面的头文件。
    #include “ “ 引用的是你程序目录的相对路径中的头文件，如果在程序目录没有找到引用的头文件则到编译器的类库路径的目录下找该头文件。

  21. 内存管理
    -这个我在使用指针的时候经常忘记
    -void calloc(int num, int size);//在内存中动态地分配 num 个长度为 size 的连续空间，并将每一个字节都初始化为 0。所以它的结果是分配了 numsize 个字节长度的内存空间，并且每个字节的值都是0。
    -void free(void *address);//该函数释放 address 所指向的内存块,释放的是动态分配的内存空间。
    -void *malloc(int num);//在堆区分配一块指定大小的内存空间，用来存放数据。这块内存空间在函数执行完成后不会被初始化，它们的值是未知的。
    -void *realloc(void *address, int newsize);//该函数重新分配内存，把内存扩展到 newsize。
    -注意：void * 类型表示未确定类型的指针。C、C++ 规定 void * 类型可以通过类型转换强制转换为任何其它类型的指针。
    -char *description;
    -description = (char *)malloc( 200 * sizeof(char) );
